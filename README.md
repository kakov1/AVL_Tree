# AVL_Tree
Реализация бинарного дерева поиска с алгоритмом балансировки с помощью AVL-инварианта.
# Сборка проекта
```
git clone https://github.com/kakov1/AVL_Tree.git
cd AVL_Tree
cmake -S . -B build/
cd build/
make
```
# Запуск основной программы
```
cd src/
./tree
```
# Запуск тестов
```
cd tests/
./test_tree
```
